# Content 模块

内容创作流程：选题 → 研究 → 草稿 → 发布。

## 流程

1. **选题** — 「我有一个想法」→ 追加到 `ideas.jsonl`
2. **研究** — 检索 `knowledge/` 已有素材（先查后写），必要时补充检索
3. **草稿** — 加载 `personal-brand/voice-guide.md` + `templates/` 对应模板
4. **发布** — 按渠道导出

## 文件

| 文件 / 目录 | 用途 |
|------------|------|
| `ideas.jsonl` | 选题池（仅追加） |
| `01-草稿箱/` | 工作中的草稿 |
| `02-素材库/` | 案例和素材 |
| `03-方法论/` | 方法论沉淀 |

## 规则

- 写内容时**只**加载 Personal Brand + Content + Templates，绝不读 Network 数据
- 用作者的声音（见 `personal-brand/voice-guide.md`），不堆砌辞藻
